### üîπ Step 1: Open Control Panel

### üîπ Step 5: Open Advanced Power Settings

### üîπ Step 6: Locate Battery Settings

### üîπ Step 7: Configure Battery Warning Behavior

  - **On** ‚Üí Shows both popup and notification (default Windows behavior)

- **Low battery level**
  - Set to **40%** (or any value you prefer)
![](website/content/blogs/windows-low-battery-alert-40-percent/img/windows-low-battery-alert-40-percent-20260222141821849.png)
Apply the same settings under:
- **On battery**
- (Optional) **Plugged in**

‚ö†Ô∏è **Note:** Windows does not provide a native option for *popup-only* alerts.

### üîπ Step 8: Save the Settings
